
# 🚀 IntelliResume – Resume Builder with Job Predictor

IntelliResume is a modern web application that helps users create professional resumes and predict suitable job roles based on their skills. It provides a clean UI, live preview, and PDF download functionality.

## ✨ Features
- 📝 Dynamic Resume Builder  
- 👀 Live Resume Preview  
- 📄 Download Resume as PDF  
- ➕ Add Multiple Skills & Projects  
- 🤖 Smart Job Predictor (based on skills)  
- 💾 Save Data using LocalStorage  
- 🌙 Dark / Light Mode Toggle  
- ✏️ Editable Resume Preview  
- 🔗 Clickable Email, GitHub & LinkedIn Links  

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript  
- html2pdf.js

## 🚀 How to Run
1. Clone the repository:
   git clone https://github.com/your-username/IntelliResume.git  

2. Open the folder:
   cd IntelliResume  

3. Run the project:
   Open index.html in your browser
